# 🏥 Sistema Integral de Gestión y Reportes Médicos

Solución tecnológica desarrollada para la gestión de registros clínicos, modelado relacional de bases de datos y automatización de reportes analíticos de recaudación.

## 🛠️ Tecnologías Utilizadas
* **Base de Datos:** MySQL (DDL / DML)
* **Lenguaje:** Python 3
* **Librerías:** `mysql.connector`, `matplotlib`

## ⚙️ Funcionalidades Principales
* **Modelado Relacional:** Diseño e implementación de esquemas con restricciones de integridad.
* **Lógica en Servidor:** Creación de *Vistas* y *Procedimientos Almacenados (Stored Procedures)* para el cálculo automatizado de ingresos y rendimiento médico.
* **Pipeline de Datos:** Script en Python (`grafico.py`) para la conexión, extracción transaccional de datos y generación de visualizaciones dinámicas.
